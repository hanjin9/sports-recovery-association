# 스포츠 회복 관리사 협회 웹사이트 디자인 전략

## 1. 벤치마킹 분석 결과

### 헬스케어/웰니스 웹사이트 핵심 요소
- **신뢰감 강조**: 의료 전문성을 시각적으로 표현 (깔끔한 레이아웃, 명확한 정보 구조)
- **명확한 가치 제안**: Hero 섹션에서 협회의 핵심 가치 즉시 전달
- **사용자 신뢰 구축**: 전문가 소개, 자격증, 고객 후기 표시
- **접근성 우선**: 반응형 디자인, 빠른 로딩, 명확한 네비게이션
- **행동 유도 (CTA)**: 예약, 상담, 연락처 등 명확한 버튼 배치

### 마사지/물리치료 웹사이트 트렌드
- **따뜻하고 친근한 색상**: 그린, 테라코타, 소프트 블루 조합
- **자연스러운 이미지**: 사람, 자연, 치료 과정 시각화
- **서비스 카드형 레이아웃**: 각 술기/서비스를 명확하게 분류
- **프로세스 시각화**: 치료 단계를 단계별로 표현

## 2. 선택된 디자인 철학

### 색상 팔레트 (럭셔리 + 신뢰감)
- **Primary**: 깊은 네이비 블루 (#1B3A5C) - 신뢰감, 전문성
- **Accent**: 골드 (#D4AF37) - 프리미엄 느낌, 강조
- **Secondary**: 소프트 테라코타 (#D97757) - 따뜻함, 인간미
- **Background**: 크림 화이트 (#F9F7F4) - 깨끗함, 고급스러움

### 타이포그래피
- **Display**: Playfair Display (세리프) - 고급스러운 제목
- **Body**: Inter (산세리프) - 명확한 본문

### 레이아웃 구조
1. **Header/Navigation**: 고정형, 로고 + 메뉴 + CTA 버튼
2. **Hero Section**: 큰 이미지 + 협회 소개 텍스트
3. **Services Section**: 3-4개 술기 카드 그리드
4. **About Section**: 협회 소개 + 회장 정보
5. **Contact Section**: 연락처, 위치, 예약 폼
